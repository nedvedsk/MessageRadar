# MessageRadar
MessageRadar is an enhanced chat monitoring and notification addon for World of Warcraft. Built upon the foundation of CChatNotifier, this version introduces advanced filtering capabilities and data management tools to help players efficiently track critical information in a crowded chat environment.

Key Enhancements over CChatNotifier
- Advanced Syntax Support: Offers more robust pattern matching and flexible syntax, allowing for precise filtering that goes beyond simple keyword matching.
- Configuration Import/Export: Easily back up, restore, or share your keyword profiles and settings via simple text strings—a feature missing in the original version.

Core Features
- Real-time Monitoring: Scans public channels, whispers, and group chats for specific triggers.
- Automated Forwarding: Redirects filtered messages to a dedicated notification window or custom channel.

MessageRadar (信息雷达)
MessageRadar 是一款专为魔兽世界设计的增强型聊天监控与通知插件。本作基于 CChatNotifier 开发，旨在通过更强大的过滤语法和便捷的数据管理功能，帮助玩家在海量信息中精准捕捉关键动态。

相比原版的差异化改进
- 更强大的语法支持：支持更复杂的匹配逻辑与语法（如增强型匹配模式），提供远超简单关键词过滤的精准度。
- 配置导入与导出：新增了配置字符串功能。玩家现在可以轻松备份、还原或与好友分享自己的关键词过滤方案。

核心功能
- 实时监控：自动扫描世界频道、密语及各类频道中的特定触发词。
- 自动转发：将匹配到的信息转发至独立的通知窗口或指定频道，防止重要消息被刷屏覆盖。
